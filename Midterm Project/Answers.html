Question: 22

What is the difference between inline and block HTML elements

Answer: A block-level element is an HTML element that begins a new line on a web page and extends the full width of the available horizontal space of its parent element. Inline element can begin within a line and doesn't start a new line. Its width only extends as far as it is defined by its tags.

Question: 23

Question: 24

Explain how a using(){ } block functions and what types of objects it applies to

Question: 25

I often drone on about HTML being a semantic language. What do I mean, and how does this affect the way HTML should be written

Answer: HTML is a semantic language because it's all about meanings and logic. Different tags or elements have different meaning. There's no one tag or element can be used for every logic and meaning.
